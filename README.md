# 🤖 Bot Discord Multi-Funcional

Um bot completo para Discord com sistema de moderação, gamificação, loja, IA e muito mais.

---

## ✨ Recursos

| Categoria | Recursos |
|-----------|----------|
| 🛡️ **Moderação** | Warns, filtros (palavras, links, spam, CAPS), logs, anti-raid, captcha, lock |
| ⭐ **Gamificação** | XP por mensagens, níveis, ranks, daily, transferência de XP |
| 🛒 **Loja** | Compra de itens com XP (fundos, avatares, cargos) |
| 🤖 **Inteligência Artificial** | Chat com IA (OpenAI, Claude, Google e mais) |
| ⬇️ **Downloads** | Baixar áudio de YouTube, TikTok, Instagram, Twitter |
| 🎉 **Diversão** | Enquetes, dados, eventos |
| 📊 **Utilidades** | Remover fundo de imagens, ping, crescimento |

---

## 🚀 Começando

### 1. Convide o bot ao seu servidor

### 2. Configure as variáveis de ambiente
```
DISCORD_TOKEN=seu_token
SUPABASE_URL=sua_url
SUPABASE_KEY=sua_chave
```

### 3. Use os comandos

---

## 📝 Comandos

### 🛠️ Administrativos
| Comando | Descrição |
|---------|-----------|
| `/painel` | Abrir painel de configuração |
| `/registrar` | Registrar o servidor |
| `/sync` | Sincronizar comandos |

### 🎮 Comandos Gerais
| Comando | Descrição |
|---------|-----------|
| `/perfil` | Ver seu nível e XP |
| `/daily` | Coletar XP diário |
| `/rank` | Ver ranking de XP |
| `/loja` | Ver itens da loja |
| `/inventario` | Ver seus itens |
| `/dado` | Rolar um dado |
| `/enquete` | Criar enquete |
| `/ping` | Ver latência |

### 🔧 Moderação
| Comando | Descrição |
|---------|-----------|
| `/warn` | Advertir membro |
| `/limpar` | Apagar mensagens |
| `/lock` | Bloquear canal |

---

## 🎛️ Painel de Controle

Digite `/painel` (requer admin) para configurar:

- **Moderação** - Warns, filtros, captcha, logs, anti-raid
- **Gamificação** - XP, level up, daily
- **Loja** - Adicionar itens
- **IA** - Configurar API
- **Auto-Role** - Cargos automáticos

---

## 📊 Requisitos

- Python 3.11+
- Supabase (banco de dados)
- discord.py
